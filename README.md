# Beware of those
1. Put each file in the exact location that you find it in starting from your home directory ~; for example you found /hyprland.lua inside .config/hypr/ put it in the location ~/.config/hypr/

2. Make sure to install the dependencies in the dependencies file that is provided in the theme you want, however some of them are optional you can delete them (not recommended) those are indicated by an *, and an # for what you can replace and be fine just make sure to change the cpnfig file accordingly.

3. There is a coresponding waybar and rofi themes in my account (repositories), organized by numbers, just take the same number in all of the three to get a better experience, although you absolutely can choose them differently, just make sure to install the dependencies as mensioned above.

4. If hyprland complains about some keybinding stuff, run wev and press the key to bind you want to bind and mark the "key:" number, then go to config file and change the problematic key to code:here_put_the_previous_number; for example if hyprland didn't recognize " + Plus" change it to " + code:86"
